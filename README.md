# Minimalist Portfolio

A minimal, performance-focused portfolio built with vanilla HTML and CSS.

## Overview
This project is a static portfolio designed to showcase projects and skills. It uses semantic markup and modern CSS without heavy frameworks.

## Features
- **Zero Dependencies**: Built entirely with standard HTML5 and CSS3.
- **Responsive Design**: Fully responsive layout that adapts to mobile and desktop screens.
- **Dark Mode**: Sleek, dark-themed aesthetic with vibrant accent colors.
- **Performance**: Optimized for instant load times and smooth animations.
- **Semantic HTML**: Accessible structure using proper HTML tags.

## Project Structure
- `index.html`: The main landing page.
- `mylms.html`: Project details page for the MyLMS Rewrite.
- `portfolio.html`: Project details page for this portfolio.
- `images/`: Directory for project assets (to be added).

## Usage
Simply open `index.html` in any modern web browser to view the portfolio.
